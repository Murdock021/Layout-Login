Bibliotecas Python 

NumPy: Manipulação de arrays e operações matemáticas complexas.
Instalação via pip: pip install numpy

Pandas: Análise e manipulação de dados, oferecendo estruturas de dados poderosas.
Instalação via pip: pip install pandas

Matplotlib: Criação de gráficos e visualizações de dados.
Instalação via pip: pip install matplotlib

Scikit-learn: Biblioteca para aprendizado de máquina e análise de dados.
Instalação via pip: pip install scikit-learn

TensorFlow: Biblioteca de aprendizado de máquina para construção e treinamento de modelos.
Instalação via pip: pip install tensorflow

PyTorch: Framework de aprendizado de máquina para criação de modelos de deep learning.
Instalação via pip: pip install torch

Django: Framework para desenvolvimento web, facilitando a criação de aplicações web.
Instalação via pip: pip install django

Flask: Framework web para construção de aplicações web simples e flexíveis.
Instalação via pip: pip install flask

Requests: Biblioteca HTTP para fazer requisições web.
Instalação via pip: pip install requests

Beautiful Soup: Análise de documentos HTML e XML, útil para web scraping.
Instalação via pip: pip install beautifulsoup4

PyAutoGUI é uma biblioteca Python usada para automatizar.Ela permite automatizar tarefas como clicar em botões, movimentar o mouse, pressionar teclas do teclado e muito mais. 
Istalação via pip: pip install pyautogui

O PySimpleGUI é uma biblioteca Python que oferece uma maneira simples e intuitiva de criar interfaces gráficas de usuário (GUI). Ele foi projetado para ser fácil de aprender e usar, permitindo que os desenvolvedores criem interfaces gráficas de forma rápida, mesmo sem muita experiência em programação de GUI.
pip install PySimpleGUI
